# プライバシーポリシー (v3)

最終更新: 2026 年 6 月 4 日 (v3 改定)

株式会社 CORE (以下「当社」) は、CORE Prism / CORE Iris (以下総称して「本サービス」) における
個人情報および利用者データの取り扱いについて、以下のとおり定めます。

## 1. 取得する情報

### 1-1. ユーザーがご自身で提供する情報
- 氏名 / ニックネーム / メールアドレス
- 業種 / 役割 / 法人名 (任意)
- 決済情報 (Stripe を経由。当社サーバーにはカード番号本体を保存しません)
- フィードバック / 改善提案 / 解約理由
- ナレッジベースへアップロードした文書・画像

### 1-2. 端末側のローカルデータ
- 端末 ID (匿名 UUID。リテンション計測のために生成。氏名等とは紐付きません)
- ペルソナ別タスク / CRM Deals / 投稿キュー / フォロワー履歴 (Iris) など
- AI モデル使用統計 (トークン数 / 呼び出し回数)
- オンボーディング完了 funnel
- ローカルキャッシュ (LP 訪問回数 / 朝ブリーフ表示済 / CTA A/B variant 等)

> ※ 上記 1-2 は原則としてユーザーの端末 (ブラウザの localStorage) に保存され、
> 当社サーバーには送信されません。ただし下記 1-3 の「集計に必要な最小値」のみ
> 匿名化されてサーバーへ送られます。

### 1-3. サーバー集計用のビーコン (匿名)
v3 で明示します。下記のごく少量のデータが当社サーバー (Vercel / Upstash) に送信されます:

- オンボーディング step 到達カウント (氏名・メアドなし)
- DAU カウント用の匿名端末 ID (UUID。電話番号 / 氏名等と紐付かない)
- フロントエンドの JavaScript エラー (メッセージ + 1 階層のスタック)
- AI モデル別の呼び出し回数 / トークン数 (本文は保存しません)
- Checkout CTA A/B テストの インプレッション / クリック (variant id のみ)
- 改善提案 (DDD) と解約理由 (EEE) — 任意で送信されるテキスト

これらは Upstash Redis に **60〜100 日の TTL** で保存され、期限後に自動削除されます。

### 1-4. Push 通知 (Web Push)
v3 で追加。ユーザーが Push 通知を **明示的に許可した場合のみ** 以下を保存します:
- ブラウザ発行の Push Subscription (endpoint + p256dh + auth)
- 端末 UA (200 文字まで)
- 登録時刻

通知配信は VAPID (公開鍵 + 秘密鍵) で署名されます。Push を許可しない場合、
当社からブラウザへの通知は一切送信されません。

## 2. 利用目的

- 本サービスの提供 / 維持 / 改善
- ユーザー サポート (お問い合わせ対応)
- 不正利用の検知 / 防止
- 利用統計に基づくプロダクト改善 (匿名化 + 集計のみ)
- メールによる通知 (請求 / 解約 / 無料体験リマインド / 再エンゲージメント)
- 法令に基づく対応

## 3. 第三者提供

当社は、以下の事業者にサービス提供上必要な範囲で情報を共有します。
- **Stripe, Inc.** — 決済処理
- **Anthropic / Google (Gemini)** — AI 応答生成 (会話本文は当社で保存しません)
- **Resend** — トランザクションメール送信
- **Upstash** — 上記 1-3 / 1-4 の集計データ保存
- **Vercel** — Web ホスティング / Edge Functions / Logs

それ以外の第三者へ個人情報を提供することはありません (法令に基づく場合を除く)。

## 4. データの保存期間

| データ | 保存場所 | 期間 |
|---|---|---|
| アカウント情報 (email 等) | Stripe Customer | 退会まで |
| ナレッジ / タスク / CRM | 端末 localStorage | ユーザーが削除するまで |
| 端末 ID | Upstash | 60 日 (アクセス無しで自動削除) |
| エラー / 使用統計 | Upstash | 35-60 日 |
| Push 購読 | Upstash | ユーザーが unsubscribe するまで |
| フィードバック | Upstash | 60 日 + Vercel Logs (90 日) |

## 5. ユーザーの権利

- **アクセス** — 自分の Stripe Customer 情報・購読状況は `/billing` から確認できます
- **訂正** — メールアドレス等は `/billing` から変更できます
- **削除** — 「すべて消す」 (BillingDashboard) で localStorage を完全消去できます
- **解約** — ワンタップで停止 (期間末まで利用可能)
- **異議申立** — 下記連絡先へお問い合わせください

## 5-2. Instagram / Meta Platform 連携について

CORE Iris は、利用者ご本人の同意のもと **Instagram API (Instagram Login)** を用いて、ご本人の Instagram プロアカウント情報を取得します。

- **取得するデータ**: プロフィール（ユーザー名・フォロワー数・投稿数・プロフィール画像）、メディア情報（各投稿のいいね数・コメント数・投稿日時・種別）、インサイト（リーチ・保存・フォロワー属性等）、および機能利用時のコメント／メッセージ。
- **利用目的**: 取得したデータは、**利用者ご本人のダッシュボード上でのみ**、分析の可視化・「伸ばす提案」・投稿支援のために使用します。**第三者への提供・販売・広告ターゲティングには一切使用しません。**
- **保存と保持期間**: アクセストークンはサーバ側で暗号化して保持し、有効期限（最長60日）経過後に失効します。プロフィールの数値は利用者の端末（localStorage）に保存され、連携解除で削除されます。
- **削除**: アプリ内で **Instagram 連携を解除** すると、保存中のアクセストークンを破棄し、取得データを削除します。手順の詳細は[データ削除の手順](https://core-prism-app.vercel.app/data-deletion.html)をご覧ください。
- **Meta 規約の遵守**: 本連携は [Meta Platform Terms](https://developers.facebook.com/terms/) および Developer Policies に従って運用します。利用者はいつでも Instagram の「設定 → アプリとウェブサイト」から本アプリのアクセスを取り消せます。

## 6. Cookie / トラッキング

当社は **Cookie ベースの広告ネットワーク (Google Ads / FB Pixel 等) を使用していません**。
利用される localStorage / sessionStorage / IndexedDB は本サービスの動作に必要なもののみです。

## 7. 子供のプライバシー

本サービスは 13 歳未満を対象としていません。13 歳未満の利用者からの個人情報取得は行いません。
