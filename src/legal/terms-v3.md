# 利用規約 (v3)

最終更新: 2026 年 6 月 4 日 (v3 改定)

株式会社 CORE (以下「当社」) が提供する CORE Prism / CORE Iris (以下総称して「本サービス」) を
ご利用いただく前に、本規約をお読みください。本サービスを使用することで、本規約に同意したものとみなされます。

## 1. サービス概要

本サービスは、AI 役員 (Claude / Gemini) を活用して、経営判断・営業・財務・マーケティングなどの
業務を補助するソフトウェアサービスです。AI による出力は **参考情報** であり、最終判断は
ユーザーご自身の責任で行ってください。

## 2. アカウント

- 13 歳以上の方のみご利用いただけます
- メールアドレスとパスワードでアカウントを作成します
- 1 アカウントは原則として 1 人のユーザーに紐付きます (チーム共有は別途プランをご用意)

## 3. 料金 / 解約

### 3-1. プラン (v2 — 2026 年 6 月時点)
- **BtoC**: Light ¥3,000 / Standard ¥5,000 / Pro ¥15,000 (税抜・月額)
- **BtoB**: Entry ¥20,000 / Standard ¥30,000 / Pro ¥50,000 (税抜・月額)
- **Enterprise**: 年間 200〜400 万円 (営業相談)

### 3-2. 無料体験
- 全プラン共通で **7 日間 無料体験** を提供

### 3-3. 解約
- ユーザーは いつでも 1 タップで解約できます (`/billing` から)
- 解約後も **当該課金期間の末日まで** ご利用いただけます
- 期間中の返金は原則行いません (当社の責に帰すべき事由を除く)

## 4. 禁止事項

- リバースエンジニアリング / 自動スクレイピングなどによる不正取得
- 他者へのなりすまし / 偽情報の登録
- AI へ違法・有害な指示を与える行為 (児童ポルノ・暴力扇動・個人攻撃等)
- 商業的な再販売 (代理店契約を結ばずに本サービスを第三者へ提供する行為)
- 業界平均を著しく超える API 呼び出し (Rate limit 設定済。突破試行は即時遮断対象)

## 5. AI 応答の取扱い

- AI モデルは Claude (Anthropic) と Gemini (Google) を使用します
- 応答は学習データと統計的推論に基づくため、**事実誤認・幻覚・偏った見解** を含み得ます
- 重要な判断 (法務 / 医療 / 投資 / 雇用) は専門家への相談を強く推奨します
- 当社は AI 応答の正確性 / 完全性 / 適合性 について保証しません
