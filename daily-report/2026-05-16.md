## 3 デモシナリオ

### 1. Notion: 今週のポストモーテムをデータベースに追加
- **target**: notion / **action**: append
- **MCP ツール**: `notion-create-pages`
- **内容**: API レイテンシ増大のポストモーテムを「振り返り」DB に記録

### 2. HubSpot: 議事録から contact を作成
- **target**: hubspot / **action**: create
- **MCP ツール**: `mcp__HubSpot__manage_crm_objects { action: "create" }`
- **内容**: 商談出席者 2 名を Contact 登録し Deal を紐付け

### 3. Gmail: 未読をサマリして 5 件返信下書き
- **target**: gmail / **action**: send
- **MCP ツール**: `mcp__Gmail__create_draft`
- **内容**: 重要キーワード付き未読を優先し返信下書きを Gmail に保存
