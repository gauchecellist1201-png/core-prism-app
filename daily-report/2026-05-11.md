# 実装日報 2026-05-11

## 第5弾の1: AI 戦略コーチ (デイリー 3 回ブリーフ)

### 概要

ユーザーが手動でボタンを押して生成する TodayBrief を、**1日3回・指定時間帯に自動配信されるプロアクティブブリーフ**へ昇華させた。
朝(06〜10時)・昼(12〜14時)・夜(18〜21時)の各スロットで、ペルソナのタスク・財務・ナレッジ・体調データを踏まえたコーチブリーフを Claude API 経由で生成し、画面上部にバナー通知する。

### 設計ポイント

- **重複防止**: `shouldRefresh()` が localStorage を参照し、同日・同スロット・同ペルソナのブリーフが存在する場合は再生成をスキップ。ページリロード時は既存ブリーフをサイレントロード(バナーは非表示)。
- **同時実行防止**: `generatingRef`(useRef) で並列 API 呼び出しをブロック。
- **サイレント失敗**: `tryGenerate` の catch はエラーを握り潰し、UI クラッシュなし。
- **briefOverride パターン**: コーチブリーフが存在する場合は `proactive.latestProposal` より優先。ユーザーが手動生成を実行すると override をクリア。
- **localStorage 上限**: `saveBriefs` は最新 30 件のみ保持。
- **音声対応**: `voiceEnabled` + `onSpeak` prop 経由で `speakNatural` を呼び出し可能。
