## A. 書類スタジオ — 見積書 → 発注書 → 納品書 → 請求書 完全シリーズ

### 追加・変更ファイル

| ファイル | 変更種別 | 説明 |
|---|---|---|
| `src/types/invoice.ts` | 拡張 | `DocumentKind`, `DocumentStatus`, `BusinessDocument` 型を追加 |
| `src/hooks/useInvoices.ts` | 拡張 | `documents: BusinessDocument[]` 状態 + `createDocument`, `updateDocument`, `removeDocument`, `duplicateAsNext`, `getDocumentsForPersona`, `getDocumentsForDeal` |
| `src/components/DocumentStudio.tsx` | 新規 | 4文書タイプ (見積/発注/納品/請求) をタブで切替。一覧・作成・詳細ビュー。ステータス遷移ボタン・複製ボタン。CRM案件紐付け対応 |
| `src/components/CRMStudio.tsx` | 拡張 | 案件詳細モーダルに「関連文書」セクションを追加。`useInvoices` 経由で `dealId` が一致する書類を一覧表示 |
| `src/components/CommandPalette.tsx` | 拡張 | `ModalKey` に `'documents'` を追加。コマンドパレットから書類スタジオを開けるように |
| `src/components/QuickActions.tsx` | 拡張 | `AUTO_GROUP` に `documents: '経営'` を追加 |
| `src/components/IdentityDashboard.tsx` | 拡張 | `DocumentStudio` をインポート、`showDocument` 状態追加、Quick Actions に追加、オーバーレイ登録 |

### 実装ポイント

- `BusinessDocument` は既存の `Invoice` 型と**完全に別系統**で共存 (`KEY_DOCUMENTS = 'core_documents_v1'`)
- ステータス遷移: estimate (draft→sent→approved) / order (draft→sent) / delivery (draft→delivered) / invoice (draft→sent→paid→cancelled)
- 番号体系: `EST/ORD/DEL/INV-{SLUG}-{YYYY}-{SEQ}` (種別ごとに独立採番)
- 「複製」機能: `duplicateAsNext()` で estimate→order→delivery→invoice の連鎖複製

## B. 人物ケア — AI 1on1 履歴 + センチメント分析

### 追加・変更ファイル

| ファイル | 変更種別 | 説明 |
|---|---|---|
| `src/types/people.ts` | 新規 | `PersonRecord`, `PersonInteraction`, `InteractionType`, `SentimentType` 型定義 |
| `src/hooks/usePeople.ts` | 新規 | `localStorage` 永続化 (keys: `core_people_v1`, `core_people_interactions_v1`)。`upsertPerson`, `newPerson`, `removePerson`, `addInteraction`, `updateInteraction`, `removeInteraction`, `getForPersona`, `getInteractionsForPerson` |
| `src/lib/peopleAnalyst.ts` | 新規 | `analyzePerson()` — `enqueueClaudeCall` を使って Claude API に交流履歴を送信し、`trustTrend`, `riskFlags`, `strengths`, `concerns`, `suggestedTopics`, `summary` を返す |
| `src/components/PeopleStudio.tsx` | 新規 | 人物一覧 (検索・タグ表示)、人物詳細 (タイムライン + `recharts` BarChart センチメント分布 + AI分析ボタン)、新規追加フォーム |
| `src/components/CommandPalette.tsx` | 拡張 | `ModalKey` に `'people'` を追加。グループ `'人材'` で分類 |
| `src/components/QuickActions.tsx` | 拡張 | `AUTO_GROUP` に `people: '人材'`、`GROUP_ORDER` / `GROUP_EMOJI` に `'人材': '👥'` を追加 |
| `src/components/IdentityDashboard.tsx` | 拡張 | `PeopleStudio` をインポート、`showPeople` 状態追加、Quick Actions に追加、オーバーレイ登録 |

### 実装ポイント

- センチメント分析チャートに `recharts` の `BarChart` を使用 (既存依存)
- AI 分析は `enqueueClaudeCall` 経由でレート制限回避
- 「次の 1on1 でのトピック 3 つ」を `suggestedTopics` として表示
