## 機能詳細

### 1. IrisCollabBoard.tsx — コラボ募集ボード

- カテゴリ: コスメ / 旅 / グルメ / ファッション / フィットネス / ライフスタイル / その他
- 投稿機能: タイトル・本文・タグ・場所・時期・フォロワー規模
- 反応ボタン (💄 ✨ 🌸 👍) + チャット開始 (スレッド形式)
- AI マッチング: 自分のジャンルを設定すると「もしかして合うかも%」スコアで並び替え
- デモシードデータ 3 件 (コスメ・旅・ファッション)
- localStorage 永続化

### 2. multiAccount.ts — マルチアカウント管理

- 種別: 個人 / サブ / タイアップ / ビジネス
- プラットフォーム: Instagram / TikTok / YouTube / X
- CRUD: `addAccount` / `updateAccount` / `removeAccount` / `setActiveAccount`
- `useMultiAccount()` React Hook でコンポーネントから利用可能
- デフォルト 1 アカウントをシード

### 3. brandGuidelines.ts — ブランドガイドライン

- トーン 6 種: フレンドリー / 高級 / カジュアル / 専門家 / ポップ / ミニマル
- ブランドカラー 2 色 (Primary / Accent)
- NGワード・必須キーワード配列
- 絵文字スタイル (たっぷり / 控えめ / なし)
- 定番ハッシュタグセット
- `runStyleCheck()`: 投稿文とガイドラインを照合し Score + 違反 + 改善提案 + 修正文を返す
- `buildGuidelinePrompt()`: AI ポスト生成時にガイドラインを注入するプロンプト生成
- `useBrandGuidelines()` React Hook

### 4. IrisDashboard.tsx — 更新

- `Tab` 型に `'collab' | 'guideline'` 追加
- タブナビに `🔗 コラボ` / `🎨 ブランドガイド` 追加
- ヘッダー右上にアカウントスイッチャードロップダウン (アバター絵文字 + ハンドル表示)
- `BrandGuidelineView` インラインコンポーネント: アカウント管理 + ガイドライン編集 + ブランドコンサル
