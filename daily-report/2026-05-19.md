# 日報 2026-05-19 — Iris UI 抹本リデザイン

## 実施内容

### 1. irisDesign.ts (新規)
デザインシステム拡張ファイルを新規作成。

- **スペーススケール**: 8pt グリッド (`SPACE` トークン)
- **角丸**: 5段階 (`sm: 8px` 〜 `2xl: 36px` + `full: 9999px`)
- **影**: 4レベル (`xs / sm / md / lg`) + glow・card ファクトリ関数
- **枠線**: `hairline / soft / accent / bold` の 4 種ファクトリ
- **タイポグラフィ**: `display / headline / title1 / title2 / body1 / body2 / label / caption / overline` — 9 スケール
- **アニメーション**: `gentle / luxe / bouncy / cinematic` 4 プリセット
- **Framer Motion variants**: `FADE_UP / FADE_IN` (delay stagger 対応)
- **CTA グラデーション**: `GRAD_CTA` = ホットピンク → パープル → ゴールド
- **`ctaStyle()` ファクトリ**: size (sm/md/lg) × ghost 対応

### 2. IrisDashboard.tsx — ナビゲーション完全リビルド

#### デスクトップ (≥ 821px): 左サイドバー
- **幅 220px**、fixed、`backdrop-filter: blur(24px)` + 半透明背景
- ロゴ (Iris wordmark + CREATOR STUDIO overline) をトップに配置
- **14 タブ** をアイコン + ラベル + 未読バッジで縦並び表示
- アクティブタブ: `linear-gradient(135deg, #E1306C → #833AB4 → #FCB045)` pill
- hover: ピンクミスト背景 + 0.18s transition
- 下部: テーマ変更ボタン + トップへ戻るボタン

#### モバイル (≤ 820px): 下部 Dock
- 上部: ミニヘッダー (sticky)
- **下部ドック**: `position: fixed; bottom: 0` — `env(safe-area-inset-bottom)` 対応
- 中央 FAB (56×56): グラデーション + case-study へ遷移
- 左2: ホーム・案件 (バッジ付き)
- 右2: 戦略・メディアキット
- コンテンツエリア: `padding-bottom: 80px` でドック分確保

#### 案件バッジ
- 進行中案件が 0 件超のとき `#E1306C` バッジを案件タブに表示
- モバイルドックの案件ボタンにも同バッジ

#### ページトランジション
- `AnimatePresence + motion` で `opacity/y` フェードアップ (0.28s, luxe easing)

### 3. IrisVoiceHome.tsx — エディトリアル Pinterest ホーム

#### ① ヒーロー週間サマリーカード
- 深紫〜インクブラック背景 + Aurora halo (accent + gold グラデ)
- 日付表示 + 週のテーマ見出し (Playfair Display italic 800)
- 近い納期案件を検出してアクション提案テキストを生成
- CTA 2本: 「コンテンツ戦略を開く」+ 「下書きを書く」(案件ある時のみ)
- 全ボタン: hover で `translateY(-2px)` + box-shadow 強化

#### ② 2カラムグリッド (`auto-fit, minmax(280px, 1fr)`)
- **左カード — TODAY'S SNAPSHOT**: 4 KPI (進行中・今週締切・今月報酬・総案件) を 2×2 グリッドで表示。クリックで各タブへ遷移
- **右カード — QUICK ACCESS**: 案件精査・丸投げ編集・交渉文・美容相談 の 2×2 ショートカット

#### ③ AI チャットエリア (Gemini Voice 風)
- AI ヘッダー: overline (CAPS) + Playfair italic h2 + Cormorant サブコピー
- チャット履歴カード: Aurora ダークモード対応
- ユーザーバブル: accent → #833AB4 グラデ + box-shadow
- AI バブル: 半透明白 + soft shadow
- ローディングドット: scale + opacity アニメーション
- サジェストボタン: hover で accent ミスト背景

#### IntersectionObserver Fade-up (Framer Motion)
- `FadeUp` コンポーネント: `useInView` (once, margin: -40px) + `motion.div`
- ヒーロー: delay 0 / グリッド左: delay 0.06s / グリッド右: delay 0.12s / チャット: delay 0.18s

### 4. IrisLanding.tsx — CTA / ナビ強化

- 全 CTA ボタン: `border-radius: 999` (pill 化) + ホットピンク → パープル → ゴールド グラデ

- `.iris-lp-ghost:hover`: 半透明白 + ボーダー強化
