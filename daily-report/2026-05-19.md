# 日報 2026-05-19 — Iris UI 抹本リデザイン

## 概要
Pinterest / Instagram / Vogue / Notion を参考に、「見ていてうっとりする」水準まで Iris UI を抹本リデザインした。

---

## 実装内容

### 1. デザインシステム拡張 `src/iris/irisDesign.ts` (新規)
- **8pt グリッド** スペーススケール (`IRIS_SPACE`)
- **角丸トークン** 5段階 8/12/18/24/36px + full (`IRIS_RADIUS`)
- **影 4レベル** sm/md/lg/xl + glow 関数 + card (`IRIS_SHADOW`)
- **タイポグラフィ 9スケール** display (Playfair italic 900) 〜 caption (`IRIS_TYPE`)
- **アニメーションプリセット** gentle / luxe / bouncy / cinematic / stagger (`IRIS_MOTION`)
- **グラデーション CTA** ホットピンク→パープル→ゴールド (`IRIS_GRADIENT`)

### 2. ナビゲーション リアーキテクチャ (`IrisDashboard.tsx`)
- **デスクトップ**: 左サイドバー 220px 固定 — ロゴ・グループ折りたたみ・タブ一覧・テーマ切替・戻るボタン
- **モバイル**: 下部 Dock 64px — 5グループアイコン + 中央 FAB (つくる)
- 旧「上部 pill タブ」を廃止。モバイルのみコンパクト横スクロールタブを補助表示

### 3. エディトリアル ホームダッシュ (`IrisEditorialHome` コンポーネント)
- **ヒーローカード**: ダークグラデ背景 + 挨拶 + グラデタイトル + 3つのステータス指標
- **2カラム Masonryグリッド**: 今日の予定 / AI インサイト / Instagram ポスト / 案件マッチング
- **パラックス背景**: `position: fixed` のアクセントオーブでグロー効果
- フォント: Playfair Display italic (見出し) × Cormorant Garamond italic (サブ) × Inter (本文)

### 4. IrisVoiceHome 視覚更新
- SummaryCard をエディトリアル仕様に — アクセントライン・大きい数字・uppercase ラベル
- 入力エリアを `border-radius: 28px` + インセットシャドウでカード感向上
- マイクボタンを CTAグラデ (ホットピンク→パープル→ゴールド) + hover/tap scale
- 送信ボタンを CTAグラデに統一

### 5. IrisLanding CTA 強化
- ヒーロー CTA・スモール CTA をホットピンク→パープル→ゴールドのグラデに統一
- `border-radius: 999px` (pill 型) + lift shadow

### 6. Google Fonts 追加 (`src/index.css`)
- Playfair Display (italic 700/800/900)
- Cormorant Garamond (italic 400/600/700)
- Inter (400/500/600/700/800)

### 7. レスポンシブ CSS
- `@media (max-width: 900px)`: サイドバー非表示、Dock 表示
- `@media (min-width: 901px)`: Dock 非表示、サイドバー表示
- モバイル main の下余白を Dock 分確保
