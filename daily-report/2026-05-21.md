## 新規ファイル

### `src/iris/IrisRevenueView.tsx`
- 収入源 (案件PR / スポンサー / アフィリエイト / グッズ/コラボ / その他) を手動記録
- recharts `BarChart` による 12 ヶ月積み上げ月次グラフ
- recharts `PieChart` による今年の収入源別内訳
- サマリー: `monthlyTotal` / `yearTotal` / `topSource`
- localStorage (`iris_revenue_entries_v1`) で永続化

### `src/iris/taxHelper.ts`
- 所得税速算表による概算税額算出 (`calcIncomeTax`)
- `estimateTax()`: 収入・経費リストから課税所得・税額・カテゴリ別/月別集計を返す
- `generateTaxChecklist()`: Claude API を呼び出して確定申告準備チェックリスト (15〜25 項目) を生成
- 基礎控除 48 万円 (令和2年以降) を適用
- localStorage で収入・経費データを永続化 (`iris_tax_incomes_v1` / `iris_tax_expenses_v1`)

### `src/iris/IrisFanEngagement.tsx`
- DM / コメント / 返信の手動記録
- 誠意タグ: スーパーファン / 長期ファン / 新規 / 個人的友人
- AI (Claude) による高エンゲージメントファン TOP10 抽出
- AI による個人チャット返信テンプレ生成 (ファン名・関係・話題を反映)
- タグ別グループ表示 + ファン詳細パネル
- localStorage (`iris_fans_v1`) で永続化
