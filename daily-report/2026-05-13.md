### 技術仕様

#### 動画フォーマット
- 参照解像度: **1080×1920px** (9:16 縦型)
- Canvas プレビュー: **405×720px** (clamp(160px, 38vw, 280px) で表示縮小)
- 合計尺: **15〜60秒** (Claude が自動調整)
- エクスポート: **30fps WebM** (MediaRecorder) / PNG 連番 (フォールバック)

#### AI 機能
- `generateScript(theme, persona, settings, targetSec)` — Claude でシーン構成・フック・キャプション・ハッシュタグを JSON 生成
- `generateBgmPrompt(scenes)` — Suno / Udio 向け BGM テキストプロンプト生成
- `voiceOverScript(scenes)` — SpeechSynthesisUtterance でシーンごとに音声プレビュー

#### Canvas レンダリング
- Instagram グラデーション背景: `#833AB4 → #E1306C → #F77737`
- Playfair Display フォントでテキストオーバーレイ
- フェード / スライドトランジション (`requestAnimationFrame` ループ)
- BGM: `<audio>` + Object URL (useEffect でクリーンアップ)
