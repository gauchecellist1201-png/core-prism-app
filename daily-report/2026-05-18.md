## AnimatedAvatar 仕様

### SVG 構造 (viewBox 0 0 100 100)
- 外周グローリング (accentColor グラデーション + feGaussianBlur フィルター)
- 背景円 (radialGradient、accentColor 基調)
- 内側ダーク顔エリア
- ブランドマーク (prism: 小三角形ポリゴン / iris: 花弁パス)
- 眉 (2 本ライン、mood で Y 座標変化)
- 目 (楕円 + 瞳 + ハイライト、CSS scaleY でまばたき)
- 鼻 (subtle ドット 2 点)
- 口 (SVG path、開口度 0-1 でリップシンク)

### アニメーション

#### まばたき
- 4〜6 秒ごとに `isBlinking` state を true → 200ms 後 false
- 目グループに `scaleY(0.1)` + `transformOrigin` で自然なまばたき
- 閉: 60ms transition / 開: 140ms transition

#### リップシンク (lipsync)
- `isSpeaking` が true の間、60〜100ms ランダム間隔で `mouthOpen` (0-1) を更新
- `computeMouthPath(openLevel)` で口のSVG path を動的計算
  - 閉: `M 37 60 Q 50 63 63 60` (なだらかカーブ)
  - 開: 楕円形パス (`Z` 閉じ) + `fill: rgba(10,6,28,0.9)` で口内を暗色に

#### Mood (表情)
| mood | 眉 | 口 |
|------|------|------|
| neutral | 穏やかなアーチ | 軽いカーブ |
| thinking | 片方下がる (軽くしかめ) | フラット線 |
| happy | 高めのアーチ | スマイルカーブ |
| curious | 両眉を上げる | 軽いカーブ |

### VoiceConversation 統合
```
isSpeaking={tts.state === 'speaking'}
mood:
  processing → 'thinking'
  ai-speaking → 'happy'
  listening   → 'curious'
  otherwise   → 'neutral'
```
