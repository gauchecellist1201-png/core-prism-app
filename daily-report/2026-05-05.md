## 実装詳細

### `src/lib/youtubeImport.ts`

- `parseYouTubeUrl(url)` — youtube.com/watch?v=… / youtu.be/… / /shorts/… / /embed/ の全形式対応
- `fetchOEmbed(url)` — `https://www.youtube.com/oembed?url=...&format=json` でタイトル・投稿者・サムネ取得 (CORS OK)
- `fetchTranscript(videoId)` — youtubetranscript.com API を試行、CORS 制約で失敗時は `null` 返却
- `summarizeWithClaude(settings, transcript, meta)` — `enqueueClaudeCall` 経由で要約・章立て・引用集・アクションを JSON で生成

### `src/components/YouTubeImportStudio.tsx`

フェーズ駆動 UI:
1. **input** — URL 入力
2. **transcript** — oEmbed でサムネ・メタ表示、字幕自動取得 (失敗時は手動貼付エリア)
3. **summarizing** — AI 処理中アニメーション
4. **result** — 要約・章立て・引用・アクションのプレビュー + 「ナレッジに保存」ボタン
5. **saved** — 保存完了 → 自動クローズ
