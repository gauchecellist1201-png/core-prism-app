## VoiceCaptureStudio 機能詳細

- **Web Speech API** (`lang='ja-JP'`, `continuous=true`, `interimResults=true`) による日本語音声認識
- **フェーズ管理**: idle → recording → preview → routing → results → saving → done
- **録音中**: 16本のバー波形アニメーション + 中間結果リアルタイム表示 + 停止ボタン
- **プレビュー**: 全文確認 → 「AI振り分け」ボタン or 再録音
- **AI振り分け**: `routeVoiceMemo()` で Claude に分類依頼、confidence 降順でソート
- **結果UI**: チェックボックス (confidence ≥ 0.6 でデフォルト ON) + confidence バー表示
- **CRM**: 既存案件選択 UI (ドロップダウン) → 選択時は addActivity、未選択は createDeal
- **保存先**:
  - task → `usePersonas().addTask`
  - knowledge → `onAddKnowledgeNote` prop
  - crm → `useCRM().addActivity / createDeal`
  - expense → `useExpenses().add`
  - idea → localStorage `core_voice_ideas`
